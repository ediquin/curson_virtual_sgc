# Curso Virtual SGC - Club de Conquistadores 🏕️

Una plataforma web interactiva y moderna diseñada para facilitar el aprendizaje y la gestión del **Sistema de Gestión de Clubes (SGC)** para directivos, secretarios y consejeros del Club de Conquistadores.

## 📌 Descripción del Proyecto

Este proyecto es un catálogo virtual de videotutoriales que explica paso a paso los procesos más importantes dentro de la secretaría y administración de un Club de Conquistadores a través de la plataforma oficial del SGC.

El sitio web está diseñado pensando en la mejor experiencia de usuario (UX), ofreciendo una interfaz limpia, un modo oscuro (Dark Mode) elegante y un **Modo Cinema** para visualizar los tutoriales sin distracciones.

## ✨ Características Principales

*   **Interfaz Moderna e Institucional**: Diseño basado en los colores y emblemas oficiales de los Conquistadores, brindando un aspecto "Premium" y profesional.
*   **Modo Cinema (Lightbox)**: Los videos no se reproducen en cuadros pequeños. Al hacer clic, se abren en un modal emergente que oscurece el fondo y auto-reproduce el contenido, mostrando su título y descripción de forma clara.
*   **Diseño 100% Responsivo**: Totalmente adaptable a dispositivos móviles, tablets y computadoras de escritorio.
*   **Animaciones Suaves**: Las tarjetas de video aparecen de manera escalonada (staggered fade-in) y cuentan con efectos interactivos al pasar el cursor.
*   **Ligero y Rápido**: Desarrollado con tecnologías nativas (Vanilla HTML, CSS y JS) sin depender de pesadas librerías externas o frameworks.

## 🛠️ Tecnologías Utilizadas

*   **HTML5**: Estructura semántica.
*   **CSS3 (Vanilla)**: Variables CSS para paleta de colores, CSS Grid, Flexbox y animaciones responsivas.
*   **JavaScript (Vanilla)**: Manipulación del DOM para la creación dinámica del modo cinema (modal) y animaciones de carga.
*   **Google Fonts**: Tipografía *Outfit*.

## 📚 Contenido de los Módulos (Tutoriales)

1.  **Registrar Miembros Nuevos o Editarlos**: Proceso desde el llenado de datos hasta el historial clínico.
2.  **Asegurar Miembros**: Gestión del seguro anual de la IASD (pagos, comprobantes y activación).
3.  **Inscribirse a Eventos**: Procedimiento para comprar cupos y subir vouchers para los eventos de la Misión.
4.  **Asignar Módulos a Líderes**: Delegación de funciones en la directiva creando usuarios y contraseñas.
5.  **Gestión de Unidades**: Creación de unidades y uso de "Encuentra un Club" para evaluación de avances.
6.  **Subir Requisitos del MCI al Ranking**: Documentación y subida de evidencias para las Metas Crucialmente Importantes.
